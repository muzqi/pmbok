挣值分析公式

指标 描述 公式
PV(Planned Value) 某段时间，计划工作量，的预算价值 PV = 用时 * 每个单位用时完成量 * 每个单位的预算
示例：10天 * 2个/天 * 10万/个
EV(Earned Value) 某段时间，实际工作量，的预算价值 EV = 实际完成的单位 * 每个单位的预算
示例：50个 * 10万/个
SV(Schedule Value) 进度偏差。
  • SV > 0，进度超前；
  • SV < 0，进度落后；
  • SV === 0，进度持平。
 SV = EV - PV
SPI(Schedule Performance Index) 进度偏差 SPI = EV / PV
CV(Cost Value) 成本偏差。
  • CV > 0，成本节约；
  • CV < 0，成本超支；
  • CV === 0，成本持平。
 CV = EV - AC
CPI(Cost Performance Index) 成本偏差 CPI = EV / AC
BAC 所有工作量，的预算价值。不包括管理储备
ETC 当前进度，距离完工的剩余估算价值
1. 采取措施，按原计划 ETC = BAC - EV
2. 不采取措施，按当前情况 ETC = (BAC - EV) / CPI = BAC / CPI
EAC 按当前进度，估计项目的总花费 EAC = AC + ETC
VAC 完工偏差 VAC = BAC - EAC
